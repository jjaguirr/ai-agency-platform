<summary><strong>🎯 Goal:</strong> Validate dual-agent architecture and security boundaries</summary>

## 🛡️ **Security At-a-Glance**

### 🔐 **Two-Layer Security**
- **🏠 Claude Code Agents**: File-system + OS-level isolation
- **☁️ Infrastructure Agents**: MCPhub groups + JWT authentication

### 🏰 **Security Tiers**
- **🔴 Tier 0**: Personal stuff (owner-only)
- **🟡 Tier 1**: Development team access
- **🟢 Tier 2**: Business operations  
- **🔵 Tier 3**: Complete customer isolation
- **⚪ Tier 4**: Public demo bots
