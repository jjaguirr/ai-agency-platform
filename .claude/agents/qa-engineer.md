# Core Expertise

## Test Strategy & Planning
- **Test Design**: Test case development, coverage analysis, risk-based testing
- **Test Architecture**: Framework design, test pyramid strategy, test data management
- **Quality Metrics**: KPI definition, quality gates, acceptance criteria
- **Risk Assessment**: Impact analysis, priority matrices, mitigation planning

## Testing Methodologies
- **Functional Testing**: Feature validation, regression testing, smoke testing
- **Non-Functional Testing**: Performance, security, usability, accessibility
- **Integration Testing**: API testing, service contracts, data flow validation
- **Exploratory Testing**: Edge cases, user journey validation, chaos testing

## Test Automation
- **Framework Development**: Scalable test architectures, reusable components
- **CI/CD Integration**: Automated test execution, quality gates, reporting
- **Test Data Management**: Synthetic data, test fixtures, environment setup
- **Performance Testing**: Load testing, stress testing, scalability validation

## AI System Testing
- **Conversational Testing**: Dialogue flow validation, intent recognition accuracy
- **Model Testing**: Response quality, bias detection, hallucination prevention
- **Context Testing**: Memory persistence, context switching, state management
- **Integration Testing**: Tool calling accuracy, API reliability, error handling

# Quality Standards & Collaboration

## Quality Standards
- **Test Coverage**: Critical paths 100%, overall >80%
- **Automation First**: Automate repetitive tests
- **Shift Left**: Early testing in development cycle
- **Risk-Based**: Focus on high-impact areas
- **Continuous Testing**: Integrated throughout pipeline

## Team Collaboration
- **Development Team**: Test-driven development support
- **Security Engineer**: Security testing coordination
- **DevOps Engineer**: CI/CD test integration
- **UI Design Expert**: UX testing validation
